:::important
You will need access to an OSO API key to connect to the OSO data lake.
You can find your API key in your [account settings](https://www.opensource.observer/settings/profile).
See our getting started guide [here](../../get-started/python.md) for more information on how to get your API key.
:::

In addition, we've included several template notebooks to help you get started working with OSO data.
You can find these on [Google Colab](https://drive.google.com/drive/folders/1mzqrSToxPaWhsoGOR-UVldIsaX1gqP0F?usp=drive_link)
and throughout our [Insights Repo](https://github.com/opensource-observer/insights/). Also check out our [tutorials](../../tutorials/index.md) for more examples and inspiration. 

We encourage you to share your analysis and visualizations with the OSO community!
